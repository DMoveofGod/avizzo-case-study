Avizzo — Gamified Google Review Platform
Role: Freelance Full-Stack / UI Developer Stack: Lovable Market: Quebec / French-Canadian small businesses

The Problem
Getting customers to leave Google reviews is a constant struggle for local businesses — standard "please leave us a review" asks have low follow-through and no incentive loop.

What I Built
A gamified review platform that turns the review-asking moment into an interactive experience:

Prize Wheel Mechanic — customers spin a configurable prize wheel in exchange for leaving a Google review, turning a passive ask into an engaging moment
Fast Setup — businesses can configure their own wheel and rewards in minutes, no technical setup required
French-Language UI — built for the Quebec market, with a polished, on-brand interface end to end
Outcome
A live, production-ready SaaS tool that gives local businesses a genuinely engaging way to convert everyday customers into 5-star Google reviews.

This is a case study for a client project. Happy to walk through the build or demo the wheel mechanic on a call.
